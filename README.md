# 

1. top 명령어: 리눅스 운영체제에세사용하는 시스템 모니터링, 프로세스 관리도구
2.  특징
    1. 실시간 모니터링: 설정된 주기마다 화면을 갱신하여 시스템을 파악
    2. 리소스 점유율: 리소스 점유율을 파악하여 어떠한 프로세스가 cpu나 메모리를 과도하게 점유하여 시스템 속도에 영향을 미치는지 확인할 수 있다
    3. 프로세스 제어:프로세스를 종료하거나 우선순위를 변경하는 등의 제어가 가능하다
3. 화면: top을 실행하면 상단의 "시스템 요약 정보"와 하단의 "프로세스 목록"으로 나뉜다
    1. 시스템 요약 정보:시스템 가동시간, 평균부하, 프로세스 상태,cpu 사용률,메모리 사용량 정보를 보여준다
        1. 시스템 가동시간: 시스템이 켜진 후 얼마나 지났는지, 현재 접속자 수는 몇 명인지 표시한다
        2. 평균 부하: cpu가 처리해야 할 작업의 대기열 길이를 의미한다
        3. 프로세스 상태: 전체 프로세스의 개수와 상태별 분류를 보여준다(Running,Sleeping,Stopped,Zombie)
        4. cpu 사용률: cpu가 어디에서 많이 사용하고 있는지를 백분율로 보여준다
        5. 메모리 사용량: 메모리 사용량을 표시한다
        
    2. 프로세스 목록: 개별 프로세스들의 상세정보를 보여준다
        1. PID : Process ID, 프로세스의 고유 식별 번호
        2. USER: User Name, 해당 프로세스를 실행한 사용자 계정 이름
        3. PR: Priority, 프로세스의 실행 우선순위
        4. NI: Nice Value, 사용자가 조절 가능한 우선순위 값
        5. VIRT: Virtual Image, 프로세스가 사용 중인 가상 메모리 총량
        6. RES: Resident Size, 프로세스가 사용 중인 실제 물리 메모리 양
        7. %CPU: CPU 사용률
        8. %MEM: 메모리 점유율
        9. COMMAND: 해당 프로세스를 실행한 명령어 이름
    3. 주요 조작키
        1. cpu 사용률이 높은 순서대로 정렬
        2. 메모리 점유율이 높은 순서대로 정렬
        3. 특정 프로세스를 강제 종료
        4. top 프로그램 종료
